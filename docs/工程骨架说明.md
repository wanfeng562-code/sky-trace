## 3. 三层模块职责

### 3.1 server（后端）

- REST：`/api/v1/flights*`、`/airports*`、`/weather-grid`、`/playback`、`/places/names`、`/datahub/*`
- WebSocket：`WS /api/v1/ws/flights`（连接即 snapshot，采集后广播，心跳 ping）
- 统一采集：`unified_pipeline`（实时 / 环境+GRD / 商业+FR24 后台）
- SQLite：`flights`、`tracks`、`flight_snapshots`、`flight_details_extra`、`airports`、`place_names`
- 瓦片代理：`/api/v1/tiles/*`（MapTiler / Stadia / OpenFreeMap）

### 3.2 client（前端）

- **地图页** `/`：实时航班、枢纽、天气网格、行政区筛选、详情与时刻表
- **统计页** `/stats`：ECharts 汇总
- **回放页** `/playback`：历史快照播放、尾迹、回放航班列表
- Pinia `useFlightStore` + `useLocaleStore`；HTTP/WS 仅访问自有后端

### 3.3 desktop（可选）

- 加载 `client/dist` 或回退 `http://localhost:5173`
- 日常开发与线上演示以浏览器 + Cloudflare Pages 为主

## 4. 关键代码文件

### 4.1 后端

| 文件 | 职责 |
|------|------|
| `server/run.py` | Uvicorn 启动，读 `.env` |
| `server/app/main.py` | FastAPI、lifespan、`init_db`、pipeline |
| `server/app/services/unified_pipeline.py` | 三层采集、GRD、FR24 后台、WS 广播节流 |
| `server/app/services/flight_store.py` | 内存快照、轨迹、`INSERT OR IGNORE` tracks |
| `server/app/services/db.py` | DDL、迁移、回放、机场种子、place_names |
| `server/app/api/routes/flights.py` | 航班/机场/weather-grid |
| `server/app/api/routes/playback.py` | 历史回放帧 |
| `server/app/api/routes/places.py` | 地名同步 |
| `server/app/api/routes/map_proxy.py` | 生产环境瓦片代理 |

### 4.2 前端

| 文件 | 职责 |
|------|------|
| `client/src/layouts/AppShell.vue` | 顶栏导航、地图页指标 |
| `client/src/views/MapView.vue` | 主地图 |
| `client/src/components/map/MapSidebar.vue` | 右侧 Tab 容器 |
| `client/src/components/flight/FlightListPanel.vue` | 列表（主用） |
| `client/src/components/flight/FlightFilterPanel.vue` | 高级筛选 |
| `client/src/utils/mapFilters.ts` | 高亮 filter（`false` 隐藏） |
| `client/src/utils/flightFeatureState.ts` | 选中/标记 feature-state |
| `client/src/utils/playbackTrail.ts` | 回放尾迹增量 |
| `client/src/services/persistentCache.ts` | localStorage 缓存 |

> 根目录 `components/FlightListPanel.vue` 为历史文件；新功能以 `components/flight/` 为准。

## 5. 最小联调流程（P0）

1. 启动后端：`cd server && python run.py` → `/api/v1/health` 返回 ok  
2. 启动前端：`cd client && npm run dev` → `http://localhost:5173`  
3. 确认 WS 在线、地图有飞机点、列表可筛选  
4. （可选）回放页在运行约 5 分钟后有 `/api/v1/playback` 数据
