# Sky-Trace 数据字典与单位规范模板

- 维护人：项目负责人（后端）
- 协同人：前端负责人（UI）
- 最近更新：YYYY-MM-DD

## 2. 数据标准总览
- 坐标系：WGS84
- 时间：UTC ISO 8601
- 速度单位：knot
- 高度单位：feet
- 航向单位：degree（0-359）
- 经纬度精度：建议 6 位小数

## 3. 统一枚举规范

### 3.1 航班状态 flight_status

| 值       | 含义   | 备注             |
| -------- | ------ | ---------------- |
| enroute  | 飞行中 | 默认状态         |
| landed   | 已落地 | 终态             |
| departed | 已起飞 | 起飞后短时间     |
| unknown  | 未知   | 数据源缺失时使用 |

### 3.2 数据来源 source_type

| 值     | 含义     |
| ------ | -------- |
| live   | 实时数据 |
| replay | 回放数据 |
| mock   | 模拟数据 |

## 4. 核心实体字典

### 4.1 FlightBrief（地图简要实体）

| 字段名      | 类型   | 必填 | 示例                 | 单位/格式 | 说明         |
| ----------- | ------ | ---- | -------------------- | --------- | ------------ |
| flight_id   | string | 是   | abc123               | -         | 航班唯一标识 |
| callsign    | string | 否   | CCA1234              | -         | 航班号       |
| lat         | float  | 是   | 23.123456            | degree    | 纬度         |
| lon         | float  | 是   | 113.123456           | degree    | 经度         |
| heading     | int    | 否   | 180                  | degree    | 航向         |
| speed_kts   | int    | 否   | 420                  | knot      | 速度         |
| altitude_ft | int    | 否   | 32000                | feet      | 高度         |
| updated_at  | string | 是   | 2026-04-12T14:30:00Z | ISO 8601  | 更新时间     |
| source_type | string | 是   | live                 | enum      | 数据来源     |

### 4.2 FlightDetail（详情实体）

| 字段名            | 类型   | 必填 | 示例    | 单位/格式    | 说明         |
| ----------------- | ------ | ---- | ------- | ------------ | ------------ |
| flight_id         | string | 是   | abc123  | -            | 航班唯一标识 |
| callsign          | string | 否   | CCA1234 | -            | 航班号       |

## 5. 业务规则约束

### 5.1 数值范围（建议）
- lat: -90 到 90
- lon: -180 到 180
- heading: 0 到 359
- speed_kts: 0 到 700
- altitude_ft: 0 到 50000

### 5.2 缺失值策略
- 必填字段缺失：后端丢弃该条记录并记录 warning 日志
- 非必填字段缺失：允许返回 null
- 前端展示：null 值显示为 --

### 5.3 冲突字段策略
- 同义字段禁止并存，例如 speed 与 speed_kts
- 若历史字段必须保留，需标记 deprecated 并给出废弃日期

## 6. 命名规范
- API 字段：snake_case
- 前端内部状态：建议 camelCase，但需在映射层统一转换
- 数据库列名：snake_case

## 7. 字段映射模板（前后端）

| 后端字段    | 前端字段   | 转换规则                | 备注         |
| ----------- | ---------- | ----------------------- | ------------ |
| speed_kts   | speedKts   | snake_case -> camelCase | 前端展示速度 |
| altitude_ft | altitudeFt | snake_case -> camelCase | 前端展示高度 |
| updated_at  | updatedAt  | snake_case -> camelCase | 刷新时间     |
