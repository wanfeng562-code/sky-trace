# Sky-Trace 数据字典与单位规范

> 字段以 `server/app/models/schemas.py` 与 [后端数据可视化与字段说明.md](./后端数据可视化与字段说明.md) 为准（2026-05-18）。下文模板条目仍适用。

## 1. 文档信息
- 文档版本：v0.3.0
- 维护人：项目负责人（后端）
- 协同人：前端负责人（UI）
- 最近更新：2026-05-12

## 2. 数据标准总览
- 坐标系：WGS84
- 时间：UTC ISO 8601
- 速度单位：knot
- 高度单位：feet
- 航向单位：degree（0-359）
- 经纬度精度：建议 6 位小数

## 3. 统一枚举规范

### 3.1 航班状态 flight_status

| 值       | 含义   | 备注             |
| -------- | ------ | ---------------- |
| enroute  | 飞行中 | 默认状态         |
| landed   | 已落地 | 终态             |
| departed | 已起飞 | 起飞后短时间     |
| unknown  | 未知   | 数据源缺失时使用 |

### 3.2 数据来源 source_type

| 值     | 含义     |
| ------ | -------- |
| live   | 实时数据 |
| replay | 回放数据 |
| mock   | 模拟数据 |

### 3.3 飞机类别 aircraft_category（OpenSky extended=1）

| 值  | 含义                 |
| --- | -------------------- |
| 0   | 未知（No info）      |
| 1   | 无人机 / 轻型        |
| 2   | 轻型（Light）        |
| 3   | 小型（Small）        |
| 4   | 大型（Large）        |
| 5   | 高涡流大型           |
| 6   | 重型（Heavy）        |
| 7   | 高性能 + 高速        |
| 8   | 旋翼机（Rotorcraft） |
| 9   | 滑翔机               |
| 10  | 轻型气球             |
| 11  | 跳伞                 |
| 12  | 超轻型               |
| 13  | 预留                 |
| 14  | 无人机（UAV）        |
| 15  | 太空运载             |
| 16  | 超重型               |

| speed_kts         | int    | 否   | 420                  | knot      | 速度                                |
| altitude_ft       | int    | 否   | 32000                | feet      | 高度                                |
| aircraft_category | int    | 否   | 4                    | enum      | OpenSky extended 飞机类别（见 3.3） |
| departure_airport | string | 否   | PEK                  | IATA      | 出发机场（商业层富集）              |

### 4.3 TrackPoint（轨迹点）

| 字段名      | 类型   | 必填 | 示例                 | 单位/格式 | 说明     |
| ----------- | ------ | ---- | -------------------- | --------- | -------- |

### 4.4 AirQualityHub（空气质量枢纽实体）

| 字段名 | 类型   | 必填 | 示例   | 单位/格式 | 说明                            |
| ------ | ------ | ---- | ------ | --------- | ------------------------------- |
| iata   | string | 是   | CAN    | IATA      | 机场 IATA 代码                  |
| lat    | float  | 是   | 23.39  | degree    | 机场纬度                        |
| lon    | float  | 是   | 113.30 | degree    | 机场经度                        |
| aqi    | int    | 是   | 3      | 1~5       | 空气质量指数（OpenWeather AQI） |
| pm2_5  | float  | 否   | 35.2   | µg/m³     | PM2.5 浓度                      |
| pm10   | float  | 否   | 48.7   | µg/m³     | PM10 浓度                       |

> **AQI 等级（OpenWeather）**：1=优良（绿）/ 2=良好（黄）/ 3=轻度污染（橙）/ 4=中度污染（红）/ 5=重度污染（紫）

## 5. 业务规则约束

### 5.1 数值范围（建议）
- lat: -90 到 90
- lon: -180 到 180
- heading: 0 到 359
- speed_kts: 0 到 700
- altitude_ft: 0 到 50000

### 5.2 缺失值策略
- 必填字段缺失：后端丢弃该条记录并记录 warning 日志
- 非必填字段缺失：允许返回 null
- 前端展示：null 值显示为 --

### 5.3 冲突字段策略
- 同义字段禁止并存，例如 speed 与 speed_kts
- 若历史字段必须保留，需标记 deprecated 并给出废弃日期

## 6. 命名规范
- API 字段：snake_case
- 前端内部状态：建议 camelCase，但需在映射层统一转换
- 数据库列名：snake_case

## 7. 字段映射模板（前后端）
