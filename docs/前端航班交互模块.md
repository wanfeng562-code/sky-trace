## 2. 本地启动

```powershell
# 先启动后端
cd D:\Gitee\Sky-Trace\server
python run.py          # 或 python -m uvicorn app.main:app --host 127.0.0.1 --port 8000

# 再启动前端
cd D:\Gitee\Sky-Trace\client
npm install            # 首次拉取依赖
npm run dev            # Vite 开发服务器，默认 http://localhost:5173
```

`client/.env.development` 中已配置后端地址，无需修改：

```
VITE_API_BASE_URL=http://127.0.0.1:8000/api/v1
VITE_WS_URL=ws://127.0.0.1:8000/api/v1/ws/flights
```

后端接口文档（Swagger UI）：http://127.0.0.1:8000/docs

## 3. 类型定义（types/flight.ts）

当前已定义的核心类型：

### FlightBrief

WS 快照与列表条目中使用的精简类型：

```typescript
interface FlightBrief {
  flight_id: string;
  callsign?: string;
  lat: number;
  lon: number;
  heading?: number;
  speed_kts?: number;
  altitude_ft?: number;
  aircraft_category?: number;    // OpenSky extended=1，index 17；0=无数据
  departure_airport?: string;    // IATA 码
  arrival_airport?: string;      // IATA 码
  airline_iata?: string;         // 航司 IATA 前缀
  dep_time?: string;             // 计划起飞时间（ISO8601）
  arr_time?: string;             // 计划到达时间（ISO8601）
  updated_at: string;
}
```

### FlightDetail

单航班详情（点击列表或地图航班后拉取）：

```typescript
interface FlightDetail {
  flight_id: string;
  callsign?: string;
  departure_airport?: string;    // IATA
  arrival_airport?: string;      // IATA
  aircraft_type?: string;
  status: string;                // en_route / on_ground / unknown
  airline_iata?: string;
  dep_time?: string;
  arr_time?: string;
  departure_weather?: WeatherInfo | null;
  arrival_weather?: WeatherInfo | null;
  last_position: FlightBrief;
  track_points: TrackPoint[];
}
```

### AirQualityHub

AQI 叠加层数据点：

```typescript
interface AirQualityHub {
  iata: string;
  lat: number;
  lon: number;
  aqi: number;       // 1=优 2=良 3=中 4=差 5=极差（OpenWeather 标准）
  pm2_5?: number;
  pm10?: number;
}
```

### ScheduleEntry

机场时刻表条目：

```typescript
interface ScheduleEntry {
  flight_iata?: string;
  dep_iata?: string;
  arr_iata?: string;
  dep_time?: string;
  arr_time?: string;
  status?: string;
  airline_iata?: string;
}
```

## 5. Pinia Store（stores/flight.ts）

### 状态

| 状态                  | 类型                                     | 说明                                |
| --------------------- | ---------------------------------------- | ----------------------------------- |
| `flights`             | `FlightBrief[]`                          | WS 推送的全量快照（每次全量替换）   |
| `selectedFlightId`    | `string \| null`                         | 当前选中航班 ID                     |
| `selectedFlight`      | `FlightDetail \| null`                   | 详情数据（HTTP 拉取）               |
| `selectedTrackPoints` | `TrackPoint[]`                           | 轨迹点                              |
| `filterCallsign`      | `string`                                 | 呼号搜索关键词                      |
| `filterCountry`       | `string`                                 | 国家/地区筛选键（COUNTRIES 对象键） |
| `filterCountryMode`   | `'airspace' \| 'departure' \| 'arrival'` | 筛选模式                            |
| `airQualityData`      | `AirQualityHub[]`                        | AQI 叠加层数据                      |
| `showAqiLayer`        | `boolean`                                | 是否显示 AQI 叠加层                 |
| `scheduleAirport`     | `string \| null`                         | 当前查看时刻表的机场 IATA           |
| `scheduleEntries`     | `ScheduleEntry[]`                        | 时刻表条目列表                      |
| `scheduleLoading`     | `boolean`                                | 时刻表加载中标志                    |
| `wsConnected`         | `boolean`                                | WS 连接状态                         |

### 主要 Actions

| Action                           | 说明                                             |
| -------------------------------- | ------------------------------------------------ |
| `selectFlight(id)`               | 设置 selectedFlightId，拉取详情 + 轨迹           |
| `loadAirQuality()`               | 拉取 AQI 数据到 airQualityData                   |
| `loadSchedules(iata, direction)` | 拉取机场时刻表，写入 scheduleEntries             |
| `connectSocket()`                | 建立 WS 连接，接收 snapshot 事件全量更新 flights |

### WS 消息格式

```json
{
  "event": "snapshot",
  "ts": 1715500000,
  "data": [ /* FlightBrief[] */ ]
}
```

每次收到 snapshot 事件，`flights` 数组整体替换（非增量更新）。

## 6. 组件说明

### FlightDetailCard.vue

触发条件：`store.selectedFlight != null`

内容布局：
- 卡片头：航司 Logo（48px，来自 `https://airlabs.co/img/airline/m/{IATA}.png`，加载失败自动隐藏）+ 呼号 + 状态标签
- 基本信息行：速度、高度、航向
- 路线行：出发机场 → 到达机场 + 计划起飞/到达时间（`dep_time` / `arr_time`）
- 天气区块（仅枢纽机场有数据）：温度、湿度、风速、能见度
- 关闭按钮（右上角 ✕）

定位：`position: fixed`，地图右下角。

### FlightListPanel.vue

左侧侧边栏，`position: fixed, left: 0`。

- 顶部：搜索框（实时过滤 callsign）
- 国家/地区选择器 + 模式切换（空域 / 出发地 / 目的地）
- 航班列表：每行含 20px 航司 Logo 缩略图 + 呼号 + 高度（ft）
- 点击任意行 → `store.selectFlight(id)`

### SchedulePanel.vue

触发条件：`store.scheduleAirport != null`（点击地图上的机场标记时设置）

布局：
- 右侧底部浮窗（`position: absolute`，宽 300px，最大高度 55vh）
- 标题行：机场 IATA + 关闭按钮
- 离港 / 到港 Tab 切换
- 每行：航班号 / 路线（DEP→ARR）/ 计划时间 / 状态色标 / 航司 Logo

关闭：点击 ✕ → `store.scheduleAirport = null`

## 7. 视图说明

### MapView.vue

地图主视图，使用 MapLibre GL JS + OpenFreeMap liberty tiles。

已实现功能：

| 功能              | 实现方式                                                               |
| ----------------- | ---------------------------------------------------------------------- |
| 航班点渲染        | GeoJSON source + circle layer，按高度着色                              |
| 选中高亮          | 独立 selectedLayer，黄色大圆                                           |
| 轨迹线            | LineString layer，蓝色渐变                                             |
| 点击航班          | `map.on('click', layer, ...)` → `store.selectFlight()`                 |
| AQI 叠加层        | AQI 数据作 GeoJSON source，circle + symbol layers；工具栏 AQI 按钮开关 |
| AQI 色阶          | 1=`#00e400` 2=`#ffff00` 3=`#ff7e00` 4=`#ff0000` 5=`#8f3f97`            |
| 国家/地区筛选聚焦 | watch filterCountry → `map.fitBounds(COUNTRIES[key].bbox)`             |
| 机场标记          | 枢纽机场 Symbol layer；点击 → `store.scheduleAirport = iata`           |
| 挂载子组件        | FlightListPanel + FlightDetailCard + SchedulePanel                     |

### StatsView.vue

统计看板，展示：
- 总架次 / 在地 / 空中 数字卡片
- 高度段分布柱状图（by_altitude_band）
- 速度段分布柱状图（by_speed_band）
- 飞机类别分布（by_category，使用 aircraft_category 字段）
- 航司前缀 Top20

### PlaybackView.vue

历史轨迹回放视图，支持时间轴拖拽与播放控制。
