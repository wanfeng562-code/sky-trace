## 4. 主地图页（MapView.vue，`/`）

### 4.1 整体布局

```
┌──────────────────────────────────────────────────────────────────┐
│  全局导航栏                                                        │
├──────────────────────────────────────────┬───────────────────────┤
│  地图主区域                                 │  右侧上下文面板          │
│  ┌────────────────────────────────────┐  │  Tab: 航班 | 枢纽 | …   │
│  │ 工具栏 + 图层面板                    │  │  ┌─────────────────┐  │
│  ├────────────────────────────────────┤  │  │ 列表 / 筛选 / 详情 │  │
│  │ MapLibre 画布                       │  │  └─────────────────┘  │
│  │  · 底图 + 自定义图层                 │  │                       │
│  │  · 右上角：缩放 / 指南针 / 定位        │  │                       │
│  │  · 右下角：比例尺（最大约 20000 km）   │  │                       │
│  └────────────────────────────────────┘  │                       │
└──────────────────────────────────────────┴───────────────────────┘
```

地图占左侧主体；右侧面板可折叠，默认展示 **航班** 或 **枢纽** 等 Tab 内容。

### 4.2 工具栏与图层面板

- 标题、实时航班计数、WebSocket 状态
- **图层** 按钮：展开底图来源（MapTiler / Stadia / OpenFreeMap）、样式切换、叠加层开关
- 叠加层包括：AQI、PM2.5 热力、风温、航班密度、**5° 天气网格**、**枢纽机场** 等

### 4.3 地图控件

| 控件 | 位置 | 说明 |
|---|---|---|
| NavigationControl | 右上 | `+` / `−` / 指南针 |
| GeolocateControl | 右上（导航组下方） | 「我的位置」：请求定位权限后飞到约 **50 km** 视野 |
| ScaleControl | 右下 | 公制比例尺，`maxWidth: 480px`，远 zoom 可显示约 **20000 km** |
| 地图 zoom | — | `minZoom: 0`，`maxZoom: 18`，支持全球尺度浏览 |

### 4.4 图层顺序（自下而上）

底图 → 暗色对比度遮罩（仅暗色样式）→ 地形晕渲 → 天气网格 → AQI/风温等 → 航班高度晕染 → **飞机图标** → **枢纽圆点+标签** → 底图地名标注。

飞机：全图 **琥珀金 `#FFBF00`** + 暗描边；枢纽：绿/蓝/紫实心圆 + **玫瑰粉 IATA 标签**（与飞机区分）。详见 [地图可视化设计说明.md](地图可视化设计说明.md)。

### 4.5 主要自定义图层

| 图层 ID | 类型 | 功能 |
|---|---|---|
| `weather-grid-fill` / `outline` | fill / line | 5°×5° 天气网格多边形 |
| `flight-altitude-glow` | circle | 按高度晕染 |
| `flight-points` | symbol | 飞机 SVG，随航向旋转 |
| `airport-points` / `airport-labels` | circle / symbol | 枢纽（按 zoom+航班量分级显示） |
| `airport-highlight-*` | circle / symbol | 选中航班或列表选中枢纽 |
| `region-highlight-*` | fill / line | **行政区划真实边界**高亮（见 4.6） |
| `selected-flight-track-line` 等 | line / symbol | 轨迹、计划航路 |

### 4.6 地区筛选与边界高亮

在右侧 **航班** Tab → **筛选** 中可选择国家 → 省州 → 市 → 区（最多四级，中国/美国/日本/德国有嵌套数据）。

选中地区后：

1. **筛选逻辑**：按领空 bbox 或起降机场过滤航班列表（Pinia `filteredFlights`）。
2. **地图高亮**：沿 **真实行政边界** 绘制填充+描边，而非简单矩形框。
   - **中国**：阿里云 DataV 区划 GeoJSON（省 / 重点市 / 区县 adcode）。
   - **美国**：本地 `public/geo/us-states.json` 州界。
   - 无矢量数据时回退为 bbox 矩形。
3. **视野**：`fitBounds` 适配到该边界范围。

实现：`utils/regionBoundaries.ts`，图层 `region-highlight-fill` / `region-highlight-line`。

### 4.7 右侧上下文面板 Tab

| Tab | 组件 | 说明 |
|---|---|---|
| 航班 | `flight/FlightListPanel` + `FlightFilterPanel` | 搜索、虚拟列表；筛选面板可折叠 |
| 枢纽 | `HubAirportListPanel` | DB 枢纽列表、中文名、离港/到港、定位/时刻表 |
| 标记 | `MarkedFlightsPanel` | 本地标记航班（localStorage） |
| 详情 | `FlightDetailCard` | 选中航班时出现 |
| 时刻 | `SchedulePanel` | 点击枢纽或「时刻表」 |

**地图高亮**：选中/标记/悬停使用 `flightFeatureState`（feature-state），取消标记时 `mapFilters.markedFlightsFilter` 返回 `false`，避免误高亮。

### 4.8 航班列表与高级筛选

**布局要点**

- 顶部：搜索框 + **筛选** 按钮（角标显示已启用条件数）
- 一行状态 Pill：全部 / 空中 / 地面
- 可折叠 **高级筛选** 面板（分区排版，非长列下拉堆叠）

**筛选能力**

| 类别 | 选项 |
|---|---|
| 高度 | 最低–最高 ft；预设：地面 / 巡航 / 高空 |
| 航速 | 最低–最高 kts |
| 飞行类型 | 民航客机、通航、直升机、无人机、轻型（OpenSky 类别） |
| 机型 | 多选 OpenSky 类别（Large、Heavy、Rotorcraft 等） |
| 地区 | 四级联动：国家 → 省州 → 市 → 区 |
| 范围 | 领空 / 出发 / 到达 |

列表项展示：呼号、空中/地面、起降机场、高度、速度。

### 4.9 枢纽机场列表（HubAirportListPanel）

- 按航班量或 IATA 排序，支持搜索
- 卡片：IATA、中文名、繁忙档位、离港/到港/相关架次
- 详情：英文全名、离港/到港航班列表，可点进航班详情
- **地图定位**、**时刻表**

### 4.10 交互摘要

| 操作 | 行为 |
|---|---|
| 点击飞机 | 选中航班 → 详情 Tab、轨迹、起降机场高亮 |
| 点击枢纽 | 时刻表 Tab |
| 切换地区筛选 | 边界高亮 + 视野飞行 + 列表过滤 |
| 底图异常 | 顶部橙色 Fallback Toast（约 8 秒） |

## 6. 历史回放页（PlaybackView.vue，`/playback`）

- 独立 MapLibre；底图逻辑与主图一致（`mapBasemap.ts`）。
- 时间范围 + 采样间隔 → `GET /api/v1/playback`；**回放前列表**为时段内出现航班的并集（`playbackCatalog.ts`）。
- 尾迹：`playbackTrail.ts` 按帧**增量**更新 LineString，避免每帧全量重建。
- 播放/暂停、倍速、时间轴；飞机样式与主图一致（琥珀金 + feature-state 高亮）。

## 7. 全局状态（Pinia：`useFlightStore`）

| 字段 | 说明 |
|---|---|
| `flights` | 当前全量航班快照 |
| `wsOnline` | WebSocket 是否在线 |
| `selectedFlightId` / `flightDetail` / `selectedTrackPoints` | 选中航班与轨迹 |
| `searchKeyword` | 呼号搜索 |
| `filterStatus` | all / airborne / on_ground |
| `filterCountry` / `filterRegion` / `filterCity` / `filterDistrict` | 四级地区 |
| `filterCountryMode` | airspace / departure / arrival |
| `filterAltMin` / `filterAltMax` | 高度范围 ft |
| `filterSpeedMin` / `filterSpeedMax` | 航速范围 kts |
| `filterFlightType` | 飞行类型 |
| `filterAircraftCategories` | 机型多选 |
| `airports` / `airQualityData` | 机场与 AQI 缓存 |
| `scheduleAirport` / `scheduleEntries` | 时刻表 |

**数据流**

1. 挂载时 REST 拉取初始快照  
2. WebSocket 推送 `snapshot` 更新 `flights`（后端采集间隔见环境配置，通常数秒～数十秒）  
3. `filteredFlights` 为 computed，合并上述筛选条件  

**方法**：`resetAdvancedFilters()` 一键清除高级筛选。
