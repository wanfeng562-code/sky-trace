# Sky-Trace 前端页面说明文档

> 本文档介绍 Sky-Trace 前端应用的整体架构、各页面布局和功能说明。

---

## 1. 技术栈

| 技术 | 版本 | 说明 |
|---|---|---|
| Vue 3 | ^3.x | Composition API + `<script setup>` |
| TypeScript | ^5.x | 全局类型标注 |
| Vite | ^5.x | 构建工具 / 开发服务器 |
| Pinia | ^2.x | 状态管理 |
| MapLibre GL JS | ^4.7.1 | 地图渲染 |
| ECharts | ^5.x | 统计图表（仅统计页） |
| vue-virtual-scroller | ^2.x | 大列表虚拟滚动 |
| axios | ^1.x | HTTP 请求（统计页） |

│   │   ├── MapView.vue      # 主地图页（/）

## 3. 全局导航栏（App.vue）

导航栏固定在页面顶部，高度 44px，深色背景（`#1e293b`）。

| 入口 | 路径 | 说明 |
|---|---|---|
| 🗺 地图 | `/` | 主地图页，默认页 |

## 4. 主地图页（MapView.vue，路径 `/`）

### 4.1 整体布局

```
┌──────────────────────────────────────────────────────────────────┐
│  全局导航栏                                                        │
├─────────────────────────────────────┬────────────────────────────┤
│  地图区域（左）                       │  航班列表侧栏（右）           │
│  ┌───────────────────────────────┐  │  ┌──────────────────────┐  │
│  │  工具栏 (toolbar)              │  │  │  搜索框               │  │
│  ├───────────────────────────────┤  │  │  状态筛选 Tabs         │  │
│  │  地图画布 (map-canvas)         │  │  │  国家/地区筛选下拉     │  │
│  │                               │  │  │  子区域筛选下拉        │  │
│  │  ┌──────────────┐             │  │  │  虚拟滚动航班列表      │  │
│  │  │  图层控制面板  │             │  │  └──────────────────────┘  │
│  │  └──────────────┘             │  │                            │
│  │                               │  │                            │
│  │  ┌──────────────────────────┐ │  │                            │
│  │  │  航班详情卡片（叠加层）    │ │  │                            │
│  │  └──────────────────────────┘ │  │                            │
│  │  ┌──────────────────────────┐ │  │                            │
│  │  │  航班时刻表面板（叠加层） │ │  │                            │
│  │  └──────────────────────────┘ │  │                            │
│  └───────────────────────────────┘  │                            │
└─────────────────────────────────────┴────────────────────────────┘
```

地图区域占据左侧主体空间，航班列表侧栏固定在右侧，两者高度均为 `calc(100vh - 44px)`。

### 4.2 工具栏（toolbar）

位于地图区域顶部，深色背景（`#0f172a`），包含：

- **标题**：`Sky-Trace`
- **飞行计数**：实时显示当前追踪航班数量，加载中时显示 `loading...`
- **⊞ 图层** 按钮：点击展开/收起图层控制面板
- **AQI** 按钮：快捷切换 AQI 空气质量叠加层（激活时高亮）

### 4.3 图层控制面板（layer-panel）

点击工具栏"⊞ 图层"按钮展开，以 slide 动画从左上角滑出，半透明毛玻璃效果。

面板分为两个区域：

#### 地图来源（provider 切换）

显示当前可用的地图 provider 按钮：
- **MapTiler**（需要 `VITE_MAPTILER_KEY`）
- **Stadia**（需要 `VITE_STADIA_KEY`）
- **OpenFreeMap**（无需 Key，始终显示）

点击切换后立即重新加载底图并恢复所有自定义图层。

若当前地图已自动回退，面板底部显示回退提示。

#### 底图样式

根据当前 provider 显示对应的样式按钮（详见[地图页说明文档](map-configuration.md)）。

#### 叠加图层

以复选框形式控制以下叠加层的显示/隐藏：

| 选项 | 说明 |
|---|---|
| AQI 空气质量 | 主要城市机场 AQI 等级彩色圆圈及数值标注 |
| PM2.5 热力图 | 基于 AQI 数据的热力图 |
| 风速风向 | 各机场风向箭头及风速文字 |
| 温度分布 | 温度彩色点及标注 |
| 航班密度热力图 | 实时航班密度热力图 |

### 4.4 地图画布与飞行数据可视化

地图使用 **MapLibre GL JS v4** 渲染，承载多个自定义 GeoJSON 图层：

| 图层 | 类型 | 功能 |
|---|---|---|
| `flight-density` | 热力图 | 实时航班密度热力图（可控） |
| `selected-flight-point` | 圆圈 | 当前选中航班高亮圆 |
| `selected-flight-track-line` | 线 | 选中航班历史轨迹线 |
| `planned-route-line` | 线 | 计划航路线（若有数据） |
| `selected-flight-track-arrows` | 符号 | 轨迹方向箭头 |
| `flight-points` | 符号 | 所有飞机图标（SVG） |
| `airport-points` | 圆圈 | 机场圆点 |
| `airport-labels` | 符号 | 机场 IATA 代码标签 |
| `airport-highlight-points` | 圆圈 | 选中航班出发/到达机场高亮 |
| `aqi-heatmap` | 热力图 | PM2.5 热力图（可控） |
| `aqi-circles` | 圆圈 | AQI 彩色圆圈（可控） |
| `aqi-labels` | 符号 | AQI 数值标签 |
| `wind-arrows` | 符号 | 风向箭头符号（可控） |
| `wind-speed-labels` | 符号 | 风速文字（可控） |
| `temp-circles` | 圆圈 | 温度分布圆圈（可控） |
| `terrain-hillshade` | 山地阴影 | 地形阴影（始终显示） |

飞机图标使用 SVG 转 canvas 方式加载，支持旋转对齐地图（`icon-rotation-alignment: "map"`），根据高度自动选择飞行图标或地面图标。

### 4.5 航班详情卡片（FlightDetailCard）

当用户点击地图上的飞机图标或侧栏列表项时，右下角叠加显示详情卡片，包含：

- 呼号、航班号、航空公司 Logo
- 飞行状态（飞行中/地面）
- 高度（ft）、速度（kts）、航向（°）
- 出发/目的机场
- 机型、计划起降时间
- 出发地天气、目的地天气、当前位置天气（含 WeatherBlock 组件）

点击右上角"×"关闭卡片。

### 4.6 航班时刻表面板（SchedulePanel）

当用户点击机场图标时，弹出机场实时进/出港时刻表。

### 4.7 航班列表侧栏（FlightListPanel）

固定于页面右侧，功能：

- **搜索**：按呼号或航班号实时过滤
- **状态筛选**：全部 / 飞行中 / 地面 三个 Tab
- **国家/地区筛选**：下拉选择国家，可进一步选择"飞越领空"或"起降机场"过滤模式
- **省/州/地区筛选**：部分国家（如中国、美国）支持子区域下钻
- **虚拟滚动列表**：使用 `vue-virtual-scroller` 支持大量航班不卡顿（每项 56px）
- 每行显示：呼号、航空公司 Logo（来自 airlabs.co CDN）、高度、速度、飞行状态标签

### 4.8 Fallback 提示条

当底图自动回退（API Key 失效、配额用尽等），顶部居中显示橙色 toast 提示，8 秒后自动消失，可点击手动关闭。

## 5. 统计页（StatsView.vue，路径 `/stats`）

### 布局

全屏统计看板，30 秒自动刷新。

### 内容

**摘要卡片行**：
- 总追踪航班数
- 飞行中数量
- 地面数量
- 按数据源分组计数

**图表区（2×2 网格）**：
- 飞行状态分布（饼图）
- 高度分布（柱状图）
- 速度分布（柱状图）
- 前 20 航司前缀排行（横向柱状图，占两列宽）

所有图表使用 **ECharts** 渲染，深色主题，响应式自适应容器大小。

## 6. 历史回放页（PlaybackView.vue，路径 `/playback`）

### 布局

```
┌────────────────────────────────────────────────┐
│  控制栏                                          │
│  [开始时间] [结束时间] [采样间隔] [加载数据]       │
│  [▶播放] [倍速] [时间线滑块]                      │
└────────────────────────────────────────────────┘
┌────────────────────────────────────────────────┐
│  地图（MapLibre GL，OpenFreeMap liberty 底图）   │
└────────────────────────────────────────────────┘
```

### 功能

- 指定时间范围（开始/结束）和采样间隔（1/5/10/30 分钟）
- 从后端加载历史航班快照帧
- 支持播放/暂停、倍速（0.5×/1×/2×/5×/10×）
- 时间线滑块可手动拖拽跳帧
- 使用独立 MapLibre 实例（OpenFreeMap liberty，不依赖主页底图）

## 7. 全局状态（Pinia store: `useFlightStore`）

主要状态（共享于地图页和其他组件）：

| 字段 | 类型 | 说明 |
|---|---|---|
| `flights` | `FlightBrief[]` | 当前全量航班快照 |
| `loading` | `boolean` | 初始加载状态 |
| `wsOnline` | `boolean` | WebSocket 连接状态 |
| `selectedFlightId` | `string \| null` | 当前选中航班 ID |
| `selectedTrackPoints` | `TrackPoint[]` | 选中航班历史轨迹点 |
| `flightDetail` | `FlightDetail \| null` | 选中航班详情 |
| `filterStatus` | `"all" \| "airborne" \| "on_ground"` | 状态筛选 |
| `filterCountry` | `string \| null` | 国家代码筛选 |
| `filterCountryMode` | `"airspace" \| "airport"` | 领空/机场过滤模式 |
| `showAqiLayer` | `boolean` | AQI 图层开关 |
| `airports` | `AirportInfo[]` | 机场数据缓存 |
| `airQualityData` | `AirQualityHub[]` | AQI 数据缓存 |

数据流：
1. 页面挂载时通过 REST API 拉取初始航班快照
2. 之后通过 WebSocket 接收实时增量推送，自动更新 `flights`
3. 选中航班时通过 REST API 拉取详情和轨迹
